## ğŸ› ï¸ Installation

### Prerequisites

- Python 3.7+ (for backend)
- Node.js & npm (optional, if you want to use more advanced frontend tooling)
- Flask, Flask-CORS, joblib, pandas, numpy (Python packages)

### Clone the Repository

```bash
git clone https://github.com/subikshapanner/Diabetes.git
cd Diabetes
```

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   pip install flask flask-cors joblib pandas numpy
   ```
3. Ensure `diabetes_model.joblib` and `scaler.joblib` are present in the backend folder.
4. Run the Flask server:
   ```bash
   python app.py
   ```

### Frontend Setup

1. Open `frontend/index.html` directly in your web browser.
2. (Optional) Deploy with any static site hosting service or integrate with advanced tooling.
